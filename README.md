# 🎮 포켓몬 폭탄 배틀 게임

재범몬 vs 엄마몬의 폭탄 배틀 게임입니다!

## 🎯 게임 방법

1. **싸우다** 버튼을 클릭하여 재범몬이 엄마몬에게 폭탄을 던집니다
2. 10번 공격하면 엄마몬이 쓰러집니다
3. 각 공격마다 핵폭발 이펙트가 나타납니다
4. R키를 눌러서 게임을 리셋할 수 있습니다

## 🚀 게임 플레이

- **재범몬** (레벨 33) - 플레이어 포켓몬
- **엄마몬** (레벨 34) - 상대 포켓몬 (HP: 67/92)
- **폭탄 데미지**: 대략 8-12 정도 (랜덤)
- **승리 조건**: 10번 공격으로 상대방 체력을 0으로 만들기

## 🎨 특징

- 포켓몬 게임 스타일 UI
- 부드러운 애니메이션 효과

- 핵폭발 이펙트
- 체력바 시스템
- 승리 폭죽 효과
